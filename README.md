
# Workflow Engine (mini-Airflow)

A distributed, DAG-based workflow orchestration engine — a scaled-down
version of Airflow/Temporal — built to demonstrate distributed systems
concepts: leader election, at-least-once execution semantics, retry
with exponential backoff, idempotency, and real-time observability.

## Status
🚧 Work in progress — built incrementally, step by step.

## Tech Stack
- **Backend:** Java 21, Spring Boot, PostgreSQL, RabbitMQ, Redisson, Docker
- **Frontend:** React, TypeScript, React Flow, Tailwind, WebSocket
